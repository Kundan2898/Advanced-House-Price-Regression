# House Price Prediction - Advanced Regression Techniques  

## 📌 Project Overview  
This project implements advanced regression techniques for predicting house prices using the Ames Housing dataset. It includes **data preprocessing, feature engineering, model selection, and ensemble learning** to improve prediction accuracy.  

## 🚀 Features  
✔ Data Cleaning and Imputation  
✔ Feature Engineering (Transformations, Encoding, Scaling)  
✔ Model Selection with **PyCaret**  
✔ Baseline Models (**Gradient Boosting, Bayesian Ridge, LGBM, Ridge, Orthogonal Matching Pursuit**)  
✔ Bagging Ensemble for improved performance  
✔ Submission file generation  

## 🛠 Tech Stack  
- **Python**  
- **Pandas, NumPy** (Data Manipulation)  
- **Matplotlib, Seaborn** (Visualization)  
- **Scikit-Learn** (Machine Learning)  
- **PyCaret** (AutoML)  
- **LightGBM, Gradient Boosting, Ridge Regression** (Modeling)  

## 📂 Project Structure  
```

 ├── train.csv  # Training data  
 ├── test.csv  # Test data
```  

## 📊 Model Performance  
- **Gradient Boosting Regressor (GBR):** Used as the baseline model  
- **Bayesian Ridge Regression:** Alternative approach  
- **Bagging Ensemble:** Combines multiple models for better predictions  
- **Evaluation Metrics:** **Root Mean Squared Error (RMSE)**
